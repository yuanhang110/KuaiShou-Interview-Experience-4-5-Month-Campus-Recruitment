如果发现别人修改了某个缓存的数据，那么CPU就会将自己本地缓存的数据过期，然后这个CPU上执行的线程在读取那个变量的时候，就会从主内存重新加载最新的数据。

lock前缀指令 + MESI缓存一致性协议

（2）内存屏障：禁止重排序

底层就是插入了XX内存屏障，XX内存屏障，就可以保证指令不会重排
